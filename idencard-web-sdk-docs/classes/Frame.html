    Class Frame

    代表單一處理中的影格。

    • canvas：已繪入當前畫面，可供模型/驗證直接讀取。
    • validations：此影格的驗證結果（尺寸、比例、面積、位移、遮罩、人臉…）。
    • card/face：偵測到的卡片/臉部結果（可能為 null 表示本幀未偵測到）。
