Idenface

Idenface

Idenface main module. Main method is "detect" for detecting face.

Constructor

new Idenface(canvasopt)

Description:
  • Create an Idenface instance.
Example
import {Idenface} from "./idenface_bundle.min.mjs";

let idenface = new Idenface();
let webcam = document.getElementById("webcam");
enableWebcam();
webcam.addEventListener("loadeddata", detectListener);

async function detectListener() {
    if (webcam.currentTime !== lastWebcamTime) {
        lastWebcamTime = webcam.currentTime;
        let frameCanvas = getFrameCanvas();
        await idenface.detect(frameCanvas);
        if (idenface.body) {
            sendToBackend(idenface.body);
            return;
        }
    }
    window.requestAnimationFrame(detectListener);
}
Parameters:
Name Type Attributes Description
canvas HTMLCanvasElement <optional>
 The canvas element is used to display the visual representation of detection results.
